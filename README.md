# Design and Application of Intelligent Multipath Routing Based on SDN

> 基于SDN网络的智能多路路由的设计与应用

## SDN 控制器 - Ryu

使用脚本启动一个使用 Ryu 的内置 Switch 和 GUI 的控制器

```shell
$ ./start_ryu.sh
```

或者使用 `ryu-manager` 运行 python 代码文件

```shell
$ ryu-manager /ryu/ryu/app/simple_switch.py
```
